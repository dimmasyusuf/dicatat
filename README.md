## Get Started
1. Clone Repository
```bash
git clone https://github.com/dimmasyusuf/dicatat.git
```

2. Open Directory
```bash
cd dicatat
```

3. Start Server
```bash
npm run start
```

## Build for Production

- Build
```bash
npm run build
```
